# Sabores de Mi Tierra - Landing Page

## Descripción

Esta es una landing page desarrollada para **Sabores de Mi Tierra**, un restaurante que busca mostrar su menú de forma sencilla, elegante y accesible.

## Características

- **Menú interactivo**: Secciones de burritas, platos principales y golosinas
- **Diseño responsivo**: Adaptado para dispositivos móviles y de escritorio
- **Paleta de colores cálidos**: Colores tierra y dorados que reflejan la identidad del restaurante
- **Logo dinámico**: Soporte para logo personalizado con fallback a texto
- **Interactividad básica**: JavaScript para mejorar la experiencia del usuario

## Estructura del Proyecto

```
sabores-de-mi-tierra/
├── index.html          # Página principal
├── style.css           # Estilos y paleta de colores
├── script.js           # Funcionalidades JavaScript
├── images/             # Carpeta para imágenes (logo y platos)
└── README.md           # Este archivo
```

## Cómo usar

1. Coloca tu logo en la carpeta `images/` con el nombre `logo.png`
2. Agrega imágenes de los platos en la carpeta `images/`
3. Abre `index.html` en tu navegador
4. Personaliza el menú editando el contenido en `index.html`

## Personalización

- **Colores**: Edita las variables CSS en `style.css` en la sección `:root`
- **Menú**: Modifica las secciones y platos en `index.html`
- **Funcionalidades**: Extiende el JavaScript en `script.js`

## Requisitos

- Navegador web moderno
- No requiere dependencias externas
